# Smart Price Analyzer 🚀

Automated web scraper and dashboard for tracking mobile phone prices from top Georgian retailers.

### Tech Stack:
* **Backend:** Python (Playwright, SQLAlchemy)
* **Database:** PostgreSQL
* **Visualization:** Streamlit
* **Infrastructure:** Docker & Docker Compose

### Features:
* Automatic data extraction from Zoommer.ge.
* Data deduplication and storage in PostgreSQL.
* Real-time price tracking dashboard.
* Fully containerized with Docker.
